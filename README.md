Assuming you're using [catchall](/crcn/catchall), just drop-in a one liner.

```html
<html>
  <head>
    <script type="text/javascript" src="http://github.com/crcn/mesh-bugsense?apiKey=[KEY_HERE]"></script>
  </head>
  <body>
  </body>
</html>
```
